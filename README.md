<h1 align="center">Replica of Snapchat login form</h1>

This project is a simple visual replica of the Snapchat login and registration form. It is a front-end only project and does not store or send any user data to a database. Its main purpose is to demonstrate the look and feel of the Snapchat login experience, including interactive elements such as alerts and validations.

## The project includes:

- HTML structure for the login and registration form.

- CSS for styling the form, positioning elements, and replicating Snapchat's branding.

- JavaScript features including:

- SweetAlert2 for user-friendly pop-up messages.

- Password validation for the login form.

- Email/username validation.

- Form input checks to ensure fields are not empty.

- Interactive feedback when incorrect credentials are entered.

- SVG graphics for the Snapchat ghost icon.
